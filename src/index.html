Cistercian Date

Cistercian Date displays the current time using a medieval number system created by Cistercian monks. Each vertical glyph represents a complete number, showing the date and time in an elegant, space-efficient format that bridges a thousand years of human ingenuity.

The Cistercian Order

The Cistercian Order was founded in 1098 in CÃ®teaux, France, as a reform movement within the Benedictine monastic tradition. These monks, known for their architectural innovations and agricultural expertise, sought to return to a stricter interpretation of the Rule of Saint Benedict.

In the 13th century, Cistercian monks developed their unique number notation system to maintain records of their inventories, astronomical calculations, and musical notations. The system allowed them to record numbers efficiently in their manuscripts while saving valuable space on expensive parchment.

The Number System

The Cistercian numeral system is remarkably efficient: it can represent any number from 0 to 9,999 using a single symbol. Each glyph is built around a vertical line, with additional marks added to represent different values.

The system divides the vertical line into four quadrants: the right and left sides of both the top and bottom. Each quadrant represents a different place value:

  • Top right: ones (1-9)
  • Top left: tens (10-90)
  • Bottom right: hundreds (100-900)
  • Bottom left: thousands (1000-9000)

By combining these marks, scribes could efficiently represent complex numbers. For example, the number 1,234 would be written as a single symbol combining the marks for 1000, 200, 30, and 4 in their respective quadrants.
